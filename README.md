## Description

### Technologies and Architecture

Technologies that are used in this projects are:

*   MVVM
*   Room
*   ViewModel
*   Koin
*   Foreground Service
*   Retrofit
*   Coroutines
*   Kotlin
*   LiveData
*   Navigation Component
*   Single Activity
*   BroadcastReceiver
*   DiffUtils
*   ListAdapter
*   Dark Mode
